# BlinkIT Sales Analysis (EDA)

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on BlinkIT sales data to understand sales patterns across different products and outlet types.

The objective is to identify key factors that impact sales and derive meaningful business insights using data analysis techniques.

## Dataset
The dataset is provided in Excel format and contains information about:
- Item type and fat content
- Item visibility and price
- Outlet type, size, and location
- Total sales

## Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis Performed
- Data cleaning and preprocessing
- Sales analysis by item type and fat content
- Outlet-wise sales comparison
- Visualization of trends using charts and graphs

## Key Insights
- Certain item types contribute significantly more to total sales
- Low Fat products show better overall performance
- Supermarket outlets generate higher sales compared to grocery stores

## Conclusion
This analysis helps in understanding customer purchase behavior and outlet performance.  
The insights derived from this project can support better inventory planning and business decision-making.
